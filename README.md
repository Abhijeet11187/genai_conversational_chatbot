# GenAI Conversational Chatbot

## Description
This project demonstrates how to build a conversational AI chatbot using Generative AI models in Python. It focuses on maintaining context, handling user queries, and generating dynamic responses through prompt-based interactions.

## Features
- Conversational chatbot implementation
- Context-aware responses
- Prompt-based interaction
- Dynamic response generation
- Basic chat flow handling

## Tech Stack
- Python
- OpenAI API / Gemini API
- Jupyter Notebook

## Concepts Used
- **Conversational AI**: Building systems that can interact with users in natural language.
- **Context Management**: Maintaining conversation history to generate relevant responses.
- **Prompt Engineering**: Structuring inputs to guide chatbot behavior.
- **LLM Interaction**: Using AI models to generate human-like responses.
- **State Handling**: Managing chat flow and previous interactions.
- **Response Generation**: Producing meaningful replies based on user input.

## Models Used
- **Text Models (LLMs)**: Used for conversation and response generation.
  - Examples: `gpt-4`, `gpt-4o`, `gpt-5`, `gemini-pro`

## Installation Steps
1. Clone the repository:
   ```bash
   git clone <your-repo-link>
   ```
2. Navigate to the project folder:
   ```bash
   cd genai-conversational-chatbot
   ```
3. Install dependencies:
   ```bash
   pip install openai google-generativeai
   ```

## Usage
1. Add your API key(s):
   ```
   OpenAI or Gemini API key (store securely)
   ```
2. Run Jupyter Notebook:
   ```bash
   jupyter notebook
   ```
3. Open and execute:
   ```
   conversational_chatbot.ipynb
   ```
